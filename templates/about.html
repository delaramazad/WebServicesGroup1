About EspotiFly

The goal of EspotiFly is to create a mash-up webbservice consisting of different APIs.

APIs:

Aviationstack API

A flight tracker and airport timetable data web service that is JSON-based and engineered to deliver global aviation data. Information such as real-time flight status, historical data and future schedules is provided. Furthermore it offers comprehensive lookup information on airlines, airports, airplanes, routes, and more. In this project it is used to provide us with flight data wich we can combine with other APIs.

MusicBrainz API

MusicBrainz API is an interface to the MusicBrainz database. It is aimed at developers of media players, CD rippers, taggers, and other applications requiring music metadata.

Spotify API

Spotify Web API enables the creation of applications that can interact with Spotify's streaming service, such as retrieving content metadata, creating and managing playlists, or controlling playback. In this project it is used to generate playlists.

Wikimedia API Portal for Wikidata

Wikimedia offers various APIs that provide access to content and data from Wikimedia projects. In this project we primarily use Wikidata to give crossreference our IATA with countries.
